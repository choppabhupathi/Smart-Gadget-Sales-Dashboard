# Smart Gadget Sales & Inventory Dashboard 📊

This project is a comprehensive Sales and Inventory Tracker built using Microsoft Excel. It transforms raw sales data into actionable business insights.

## 🚀 Project Overview
The goal of this project is to manage gadget inventory efficiently, track sales performance, and automate stock alerts.

## 🛠 Features Implemented
- **Data Analysis:** Used core Excel functions like `SUM`, `COUNTIF`, `MAX`, and `AVERAGE` for key metrics.
- **Automated Alerts:** Implemented nested `IF` logic and **Conditional Formatting** to identify "Low Stock" and "Re-Order" products.
- **Visual Analytics:** Created interactive **Pie Charts** for category-wise revenue and **Bar Charts** for inventory levels.
- **Professional Dashboard:** A clean, summary-driven layout designed for business decision-making.

## 📈 Analytical Insight
The dashboard highlights that while the **Laptop** category generates the highest revenue, the inventory for critical products like the **Dell XPS 15** is low, requiring immediate procurement to prevent sales loss.
